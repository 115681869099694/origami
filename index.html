Origami is a free tool for designing modern user interfaces. Quickly put together a prototype, run it on your iPhone or iPad, iterate on it, and export code snippets your engineers can use.

We created Origami to help us design and build many of our products at Facebook, including Instagram, Paper, Messenger, Slingshot, Rooms, and Groups. We’re excited to see what people make with it.

Convert to Code

Export code snippets to iOS, Android, and the web with a click.

Origami can now export code for the parts of your prototypes you used to have to tediously spec to engineers. Now with the click of a button, you can deliver your animation specs as code to your engineers that they can copy-and-paste into the project.

This uses our free, open source animation frameworks to deliver the exact same feel across all platforms: Pop for iOS, Rebound for Android, and Rebound JS for the web.

Take a look at some sample code this generated.

  • iOS .m
  • Android .java
  • Web .js

Photoshop Integration

Origami lets you create links between layers in your prototype and Photoshop files. Changes to your PSDs will show up in Origami the moment you save without resetting your prototype. You can even update assets with one hand while interacting with them in Origami Live with the other.

Prototype for desktop, too

Use Origami to prototype desktop websites and apps with features like trackpad scrolling, keyboard input, FaceTime camera, and a resizable web browser frame for prototyping responsive designs.
